# 🐾 Patinhas Felizes

## 📖 Sobre o Projeto

**Patinhas Felizes** é um site institucional desenvolvido para um pet shop fictício, com o objetivo de apresentar a empresa, seus serviços e disponibilizar um canal de contato para os clientes.

Este projeto foi criado como **desafio prático do curso SCTec**, aplicando os conceitos estudados durante a formação em desenvolvimento web.

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* **HTML5** – Estruturação das páginas e conteúdo.
* **CSS3** – Estilização, layout e responsividade.
* **JavaScript** – Interatividade e manipulação do formulário de contato.

## 🎯 Objetivos do Projeto

Durante o desenvolvimento foram aplicados diversos conceitos aprendidos no curso SCTec, como:

* Estrutura semântica com HTML;
* Organização de conteúdo em seções;
* Navegação interna através de âncoras;
* Estilização com CSS;
* Uso de fontes externas (Google Fonts);
* Inserção de imagens e ícones;
* Criação de formulários com validação;
* Manipulação de eventos utilizando JavaScript.

## 💡 Aprendizados

Este projeto permitiu consolidar conhecimentos fundamentais de desenvolvimento front-end, praticando a construção de uma página completa e funcional utilizando apenas HTML, CSS e JavaScript.

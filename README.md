# 🐾 Patinhas Felizes

## 1. Identificação do Projeto

**Nome do Projeto:** Patinhas Felizes

**Autor:** Nicolas Nardelli

**Curso:** SCTec

**Tecnologias Utilizadas:**

* HTML5
* CSS3
* JavaScript

**Descrição:**
O projeto **Patinhas Felizes** consiste no desenvolvimento de um site institucional para um pet shop fictício. O objetivo é apresentar a empresa, divulgar seus serviços e disponibilizar um formulário de contato para potenciais clientes.

Este projeto foi desenvolvido como parte de um **desafio prático do curso SCTec**, aplicando os conhecimentos adquiridos durante as aulas de desenvolvimento web.

---

## 2. Etapas de Desenvolvimento

### 1. Estruturação com HTML

Foi utilizada a linguagem HTML para construir a estrutura do site, criando:

* Cabeçalho com logotipo e menu de navegação;
* Seção inicial de apresentação;
* Seção "Quem Somos";
* Seção de serviços;
* Formulário de contato;
* Rodapé.

Também foram utilizados elementos semânticos para melhorar a organização do código.

### 2. Estilização com CSS

A estilização foi realizada através do CSS, aplicando:

* Layout responsivo;
* Cores e identidade visual do pet shop;
* Organização dos conteúdos em cards;
* Formatação de textos;
* Efeitos visuais e espaçamentos;
* Integração com fontes do Google Fonts.

### 3. Implementação com JavaScript

Foi utilizado JavaScript para adicionar interatividade ao formulário de contato, permitindo o tratamento dos dados informados pelo usuário por meio da função de cadastro de lead.

---

## 3. Guia de Execução

### Requisitos

Para visualizar o projeto é necessário apenas um navegador web moderno, como:

* Google Chrome
* Microsoft Edge
* Mozilla Firefox

### Como Executar

1. Faça o download ou clone o repositório do projeto.
2. Certifique-se de que todos os arquivos estejam na mesma estrutura de pastas.
3. Abra o arquivo `index.html`.
4. O site será carregado automaticamente no navegador.

### Estrutura do Projeto

